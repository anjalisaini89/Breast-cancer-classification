# 🩺 Breast Cancer Classification using Support Vector Machine (SVM)

A machine learning project that predicts whether a breast tumor is **Benign (B)** or **Malignant (M)** using the Wisconsin Breast Cancer Dataset and the Support Vector Machine (SVM) algorithm.

---

## 📌 Features

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Missing Value Analysis
- Feature Scaling using StandardScaler
- Train-Test Split
- Support Vector Machine (SVM) Classifier
- Model Evaluation
- Confusion Matrix
- Classification Report
- Accuracy Score
- Data Visualization using Seaborn & Matplotlib

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- KaggleHub
